批量解压工具 (7zip-batch-extractor)
这是一个基于 Python 和 7-Zip 开发的图形化批量解压工具，旨在提高处理大量压缩包（尤其是带密码和分卷包）的效率。
主要功能
多线程并发：支持自定义并发任务数，充分利用多核 CPU 性能。
智能分卷识别：自动关联 .001、.part1.rar、.z01 等分卷格式，只需添加其中一个文件即可导入整个序列。
密码管理系统：
支持保存常用密码到本地列表。
支持“自动尝试”模式：解压时自动轮询已保存的密码库，无需手动逐个输入。
动态资源调度：实时监控 CPU 占用率，可根据系统负载自动调整并发任务数，防止系统卡顿。
高级过滤与输出：
支持按后缀名过滤（如：只解压 .mp4 文件）。
支持解压后自动删除原压缩包。
支持自定义输出路径或在原位置创建同名文件夹。
使用说明
下载运行：请前往本仓库的 Releases 页面下载打包好的 exe 文件。
环境要求：系统需安装 7-Zip。程序会自动搜索默认安装路径。
配置文件：程序运行后，会在 exe 所在同级目录下自动生成 saved_passwords.json 文件，用于持久化存储您的解压密码列表。
操作步骤：
点击“选择压缩包”导入任务。
在密码管理界面添加常用密码。
设置并发数，点击“开始解压”。
功能项概览
并发解压数：设置同时运行的 7z 进程数量。
自动调整 (CPU)：当 CPU 占用率超过设定阈值时，自动限制新任务启动。
只解压后缀：输入后缀名（如 .jpg,.png）可实现定向提取。
管理密码：打开密码库进行添加、删除或置顶操作。
注意事项
本工具仅为 7-Zip 的图形化封装，核心解压功能依赖于系统中安装的 7-Zip 程序。
若解压失败，请检查状态栏显示的错误信息（如：密码错误、分卷不完整等）。
