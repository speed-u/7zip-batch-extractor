# 批量解压工具 (7zip-batch-extractor)

这是一个基于 Python 和 7-Zip 开发的高效图形化批量解压工具。专为处理大量带密码、分卷压缩的任务而设计，支持自动化资源调度。

## 主要功能

1. 多线程并发解压
- 支持自定义并发任务数（默认3-50个）。
- 显著提升大量压缩包的处理速度。

2. 智能分卷识别
- 自动识别并关联 .001、.part1.rar、.z01 等分卷格式。
- 只需添加序列中的任意一个文件，程序会自动识别整个分卷组。

3. 增强型密码管理
- 密码本记忆：支持保存常用密码。
- 自动尝试：解压时可自动轮询密码本中的所有密码，无需手动逐个输入。
- 配置文件：程序运行后会在同级目录下自动生成 saved_passwords.json 文件，用于持久化存储密码。

4. 动态资源监控
- 实时显示系统 CPU 占用率。
- “自动调整”模式：基于系统负载动态启动新任务，防止解压时系统卡顿。

5. 灵活提取策略
- 支持按后缀名过滤（例如只解压 .jpg 或 .mp4 文件）。
- 支持解压后自动删除原压缩包。
- 支持创建同名子文件夹或统一解压到指定目录。

## 安装与使用

1. 下载运行
- 请前往本仓库的 Releases 页面下载打包好的 .exe 文件直接运行。

2. 环境要求
- 系统需安装 7-Zip 软件。程序会自动搜索默认安装路径（C:\Program Files\7-Zip）。

3. 操作流程
- 点击“选择压缩包”导入任务。
- （可选）在“管理密码”中添加常用解压密码，并勾选“自动尝试已保存密码”。
- （可选）可以设置并发任务数，可以勾选依据cpu占用率自动增加并发任务数
- 设置并发数及输出路径，点击“开始解压”。

## 功能说明

- 并发解压数：设置同时启动的 7z 进程数量。
- 最大 CPU%：在自动调整模式下，当系统 CPU 超过此阈值将不再启动新任务。
- 只解压后缀：输入格式如 .jpg,.png，则解压任务中仅提取指定类型文件。
- 密码文件：exe 同目录下的 saved_passwords.json 可手动备份或迁移到其他设备。

## 注意事项

- 本工具为 7-Zip 的图形化封装插件，核心解压逻辑由 7-Zip 执行。
- 若状态栏提示失败，通常原因为密码错误、磁盘空间不足或分卷包不完整。
